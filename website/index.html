Icey Client

A premium Minecraft launcher with an icy theme. Vanilla & Fabric support, mod browsing, and a beautiful UI.

Download GitHub

Windows & macOS · Requires Java 17+

What you get

One-Click Launch
Select an installation and hit play. Vanilla and Fabric versions supported out of the box.
Mod Browser
Search and install mods & resource packs from Modrinth and CurseForge directly in the launcher.
Customizable
Themes, accent colors, RAM allocation, JVM args — tailor the launcher to your setup.
Drag & Drop
Drop .jar mods and .zip resource packs straight into the launcher. They go to the right folder automatically.
Multi-Installation
Create multiple installations with different versions. Each gets its own mods, resource packs, and settings.
Icy Theme
A carefully crafted dark UI with frosted glass panels, ice blue accents, and smooth animations everywhere.
